## 1. Information We Collect

## 8. Permissions Explained

The app requests the following Android permissions:

| Permission | Purpose | When Used |
|-----------|---------|-----------|
| **WRITE_EXTERNAL_STORAGE** | Save game progress to device storage | When saving game (manual action) |
| **READ_EXTERNAL_STORAGE** | Load saved games from device storage | When loading game (manual action) |

**Evidence from Codebase:**
```python
# main.py - Permission request
def request_android_permissions():
    from android.permissions import request_permissions, Permission
    request_permissions([
        Permission.WRITE_EXTERNAL_STORAGE,
        Permission.READ_EXTERNAL_STORAGE
    ])
# Only storage permissions; no network, camera, location, etc.
```

**Why These Permissions?**
- **Write:** To save your game progress (5 save slots + quicksave)
- **Read:** To load your saved games when you resume playing

**Permissions NOT Requested:**
